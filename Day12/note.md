# DOM

## DOM Tree là mô hình cây tài liệu HTML dạng đối tượng.

## DOM có 3 node chính

### Element Node

- getElementById.
- getElementsByClassName.
- getElementsByTagName.
- querySelector.
- querySelectorAll.

### Text Node

- innerText.
- innerHTML.
- outerText.
- outerHTML.

### Attribute Node

- className, classList.
- style.
- attributeName.
- setAttribute/getAttribute/removeAttribute.

## DOM Events

Có 3 cách thêm sự kiện cho DOM

1. Thêm trực tiếp trong thẻ HTML.
2. Sử dụng thuộc tính của đối tượng DOM: onclick, onmouseover,...
3. Sử dụng phương thức addEventListener.
